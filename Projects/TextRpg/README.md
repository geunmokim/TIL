콘솔 기반으로 구현한 간단한 텍스트 RPG입니다.

사용 언어 / 기술

- 언어: C++
- 개발 환경: Visual Studio
- 핵심 기술
  - 객체지향 설계 (상속, 다형성, 가상함수) //virtual,override 등등 사용
  - 동적 메모리 관리 (new / delete)
  - 파일 입출력 (fread / fwrite) - 세이브 / 로드
  - STL (vector, iterator, for_each)
  - static_cast를 활용한 타입 캐스팅

구현한 기능

- 캐릭터 선택: 전사 / 마법사 / 도적 중 선택 (직업별 스탯 상이)
- 던전 (전투 시스템)
  - 초급 / 중급 / 고급 난이도 선택
  - 몬스터: 슬라임 / 오우거 / 골렘
  - 전투 / 도망 선택 가능
  - 승리 시 경험치 / 골드 획득, 레벨업 시 스탯 증가
  - 패배 시 골드 / 경험치 반감
- 상점 시스템
  - 초급 / 중급 / 고급 등급별 무기 / 방어구 구매
  - 인벤토리 아이템 판매 (판매가 = 구매가의 50%)
- 인벤토리 시스템
  - 아이템 장착 / 해제
  - 장착 중인 아이템 판매 불가 처리
  - 최대 5칸 슬롯 관리
- 아이템 시스템
  - 무기 (공격력 증가) / 방어구 (최대 체력 증가)
  - 장착 중 상태 표시
- 세이브 / 로드
  - 플레이어 정보 및 인벤토리 상태 저장
  - 게임 재시작 시 이어하기 가능

## 조작 방법

모든 입력은 숫자 입력 후 엔터로 진행됩니다.

메인 메뉴
```
1. 던전   2. 인벤   3. 상점   4. 저장   5. 종료
```

던전
```
1. 초급   2. 중급   3. 고급   4. 나가기
전투 중 → 1. 전투   2. 도망
```

상점
```
1. 초급   2. 중급   3. 고급   4. 판매   5. 나가기
구매 화면 → 1. 무기구매   2. 방어구구매   3. 나가기
```

인벤토리
```
1. 장착   2. 해제   3. 나가기
장착/해제 → 번호 입력 (0: 나가기)
```
